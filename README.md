# Garden Restoration Assistant

A tool to help people restore native Australian ecosystems — from a single Sydney
backyard to a bushland reserve regenerated by volunteers. It recommends what to plant
and why, grounded in ecological data and local guidance, so restoring native habitat is
less about specialist knowledge and more about getting started.

## Status

Early and actively under construction. Currently in **Phase 0**: building the data
backbone — pulling real species data from public ecological sources onto disk. The
recommendation and identification features described above are the destination, not the
current state.

## Setup

Requires [conda](https://docs.conda.io/) and git.

```bash
# 1. Clone the repository
git clone https://github.com/saloni-11/garden-restoration-assistant.git
cd garden-restoration-assistant

# 2. Create and activate the environment (Python 3.12)
conda create -n garden python=3.12
conda activate garden
```

You should see a `(garden)` prefix in your terminal once the environment is active.

More setup steps (dependencies, running the data pipeline) will be added here as those
parts of the project are built.
